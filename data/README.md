# Как подтянуть реальные данные из таблицы

Приложение работает только с внешними данными таблицы. Чтобы в поле «ФИО» были люди и появились метрики, опубликуйте таблицу через Google Apps Script или положите локальный `data/workbook.json` в формате ниже.

Минимальный формат:

```json
{
  "infoRows": [
    { "fullName": "Фамилия Имя Отчество", "role": "Роль из листа Инфо", "managerRole": "Роль руководителя из листа Инфо" }
  ],
  "metricSheets": [
    {
      "name": "Название листа",
      "rows": [
        {
          "frequency": "ежедневно",
          "metric": "Название метрики из колонки B",
          "description": "Описание из колонки C",
          "goal": "Цель из колонки D",
          "role": "Должность ответственного из колонки F",
          "managerRole": "Руководитель для дашборда из колонки J",
          "reportFormat": "Как отчитываться",
          "type": "checkbox"
        }
      ]
    }
  ]
}
```

Соответствие колонок в вашей таблице:

- лист `Инфо`: `fullName` = ФИО, `role` = роль сотрудника, `managerRole` = роль руководителя из колонки C;
- листы с метриками: `frequency` = колонка A, `metric` = название из колонки B, `description` = описание из колонки C, `goal` = цель из колонки D, `role` = должность ответственного из колонки F, `managerRole` = руководитель для дашборда из колонки J;
- допустимые значения `frequency`: `ежедневно`, `еженедельно`, `ежемесячно`.

Если данные приходят напрямую из опубликованной таблицы/API, измените `window.PULT_DATA_URL` в `src/config.js` на URL, который отдаёт JSON в таком же формате. Если URL недоступен или скрипт не опубликован для доступа по ссылке, приложение не подставляет тестовые ФИО и показывает пустой каталог.


## Вариант через Google Apps Script

Если хотите, чтобы пульт забирал данные напрямую из Google Sheets:

1. Откройте таблицу → `Расширения` → `Apps Script`.
2. Скопируйте код из `data/google-apps-script.js` в `Code.gs`.
3. В начале скрипта проверьте `CONFIG`: по умолчанию лист называется `Инфо`, ФИО берётся из колонки A, роль сотрудника — из колонки B, роль руководителя — из колонки C, а на листах метрик A = периодичность, B = название, C = описание, D = цель, F = должность ответственного, J = руководитель для дашборда.
4. Нажмите `Deploy` → `New deployment` → тип `Web app`.
5. Выберите `Execute as: Me` и `Who has access: Anyone with the link`.
6. Скопируйте URL веб-приложения и вставьте его в `src/config.js`. После изменения URL или фронтенда увеличьте версию в `package.json`, `src/version.js` и query-параметры ресурсов в `index.html`, чтобы хостинг/браузер не показывал старые `main.js` и `config.js` из кеша.

Скрипт отдаёт JSON в том же формате, который ожидает пульт: `infoRows` для листа `Инфо` и `metricSheets` для всех остальных листов.
